# EVALUACIÓN 2 - ANÁLISIS SUPERVISADO Y NO SUPERVISADO

Mediante este proyecto se pone en practica lo aprendido en las clases, aplicando modelos de **Machine Learning** bajo lo que es el **Análisis Supervisado** y **Análisis No Supervisado**.

## Python Version
- 3.10.11

## Dependencias
- Pandas
- Numpy
- Sklearn
- Seaborn
- Matplotlib

### Instrucciones del Proyecto
Clonar Proyecto: 
```
git clone "Link del Repositorio"
```
Crear Entorno Virtual
```
Python -m venv "Nombre del Entorno"
```
Ejecutar Proyectos
```
python "Nombre del Archivo".py
```
